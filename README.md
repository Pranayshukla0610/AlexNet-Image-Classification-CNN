# AlexNet-Image-Classification-CNN
## Overview
This repository demonstrates the implementation of **AlexNet**, a pioneering Convolutional Neural Network (CNN) architecture that significantly advanced deep learning for image classification. The project focuses on feature extraction, hierarchical learning, and large-scale image recognition using deep convolutional networks.

## Objectives
- Understand the architecture and components of AlexNet.
- Implement deep CNNs for image classification tasks.
- Learn feature extraction through convolutional layers.
- Evaluate model performance using standard classification metrics.

## Model Architecture

### AlexNet
The AlexNet architecture consists of:

- Input Layer
- Convolutional Layers
- ReLU Activation Functions
- Max Pooling Layers
- Dropout Layers
- Fully Connected Layers
- Softmax Output Layer

AlexNet introduced ReLU activation and dropout regularization, significantly improving training efficiency and model performance.
